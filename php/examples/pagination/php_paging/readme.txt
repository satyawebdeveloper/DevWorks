Open the config.php file and enter the mysql database user id password and db name. 
Use the sql_dump.txt file to create your student table and to populate the records.
Keep all the files in one directory and open the demo_paging1.php file from any location accessed by your server path. 
The other file demo_paging2.php has the option of selecting number of records per page by the user.
